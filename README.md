## 실행 방법

### client 실행

1. **원격 저장소 복제**

```bash
$ git clone https://github.com/sh5080/TwoFiveOne/
```

2. **프로젝트 폴더로 이동**

```bash
$ cd client
```

3. **필요한 node_modules 설치**

```bash
$ yarn install
```

4. **개발 서버 실행**

```bash
$ yarn start
```

<br />
